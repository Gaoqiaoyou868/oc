### 1.2 接线表

#### 单片机 → LCD1602 (LM016L)

| AT89C52 引脚 | LM016L 引脚 | 说明 |
|:---:|:---:|---|
| P0.0 ~ P0.7 | D0 ~ D7 | 8位数据总线 |
| P2.6 | RS (Pin4) | 寄存器选择: 0=命令 / 1=数据 |
| P2.5 | RW (Pin5) | 读写选择: 0=写 / 1=读 |
| P2.7 | E (Pin6) | 使能信号 |
| VCC | VDD (Pin2) | +5V 电源 |
| GND | VSS (Pin1) | 地 |
| GND | VEE (Pin3) | 对比度调节 (Proteus中可直接接地) |

#### 单片机 → L298 → 直流电机

| AT89C52 引脚 | L298 引脚 | L298 引脚 | 电机引脚 | 说明 |
|:---:|:---:|:---:|:---:|---|
| **P2.0** | **IN1** | **OUT1** | 电机 + | 正转控制 |
| **P2.4** | **IN2** | **OUT2** | 电机 - | 反转控制 |
| **P3.1** | **IN3** | — | — | 连接但未用(预留B桥) |
| — | **ENA** → +5V |   |   | 常使能 |
| — | **VS** → +12V |   |   | 电机供电 |
| — | **VSS** → +5V |   |   | 芯片逻辑供电 |
| — | **GND** → GND |   |   | 共地 |
| — | **SENSE_A** → GND |   |   | 不用电流检测 |
| — | IN4/ENB/OUT3/OUT4/SENSE_B |   |   | 悬空 (只用 A 桥) |

> **软件 PWM 调速原理**: L298 的 ENA 接 +5V 常使能, 单片机通过 P2.0/P2.4 输出方波:
> - `IN1=1, IN2=0` → OUT1=H, OUT2=L → 电机正转
> - `IN1=0, IN2=0` → OUT1=L, OUT2=L → 电机停止
> - 每个周期固定 8ms (125Hz), 调节导通时间控制转速

#### 单片机 → 按键 (P1 高四位, 低电平有效, 需上拉电阻)

| 按键 | AT89C52 引脚 | 上拉电阻 | 功能 |
|:---:|:---:|---|---|
| K1 | **P1.7** | 10KΩ → VCC | 启动 / 停止 |
| K2 | **P1.6** | 10KΩ → VCC | 风速 +1 |
| K3 | **P1.5** | 10KΩ → VCC | 风速 -1 |
| K4 | **P1.4** | 10KΩ → VCC | 短按=换模式, 长按2秒=设/关定时 |

> **接线方法**: 每个按键一脚接单片机引脚, 另一脚接 GND。同引脚再拉一个 10KΩ 电阻到 VCC。

#### 单片机 → LED

| LED | AT89C52 引脚 | 限流电阻 | 说明 |
|:---:|:---:|---|---|
| D1 | **P3.0** | 220Ω 串接 | 电源指示灯(运行=亮) |
| D2 | P2.1 | 220Ω 串接 | 模式指示灯(常风=常亮, 自然风/睡眠=闪烁) |

> **接线方法**: 单片机引脚 → 220Ω 电阻 → LED 阳极 → LED 阴极 → GND

#### 单片机 → 蜂鸣器

| AT89C52 引脚 | 蜂鸣器 | 说明 |
|:---:|:---:|---|
| P2.5 | +极 | 方波驱动发声 |
| GND | -极 | 接地 |

#### 单片机最小系统

| AT89C52 引脚 | 连接 | 说明 |
|:---:|:---:|---|
| XTAL1 (Pin19) | 晶振一脚 + 30pF电容 → GND | 振荡电路 |
| XTAL2 (Pin18) | 晶振另一脚 + 30pF电容 → GND | 振荡电路 |
| RST (Pin9) | 10KΩ → GND, 10uF → VCC, 按键 → VCC | 上电复位 + 手动复位 |
| EA/VPP (Pin31) | VCC | 使用内部程序存储器 |

## 二、用户操作手册

### 2.1 硬件版 (main.c — 红外遥控 + 按键)

#### 启动前准备

1. **跳线帽配置**:
   - 段选: **ON**
   - 位选: **ON**
   - **JP4(矩阵键盘): OFF** (拔掉!)
   - **JP8(直流电机+按键): ON** (插上!)
   - 原因: P1 口同时连矩阵键盘(JP4) 和外部设备(JP8), 必须二选一

2. 确保红外接收头已插在开发板相应位置
3. 确保遥控器有电池

#### 按键功能 (4个按键全部可用)

| 按键 | 功能 | 按下后反应 |
|:---:|---|---|
| **KEY1** (P1.7) | 启动/停止风扇 | 蜂鸣器响一声, 风扇启/停 |
| **KEY2** (P1.6) | 风速 +1 档 | 蜂鸣一声, 风扇加速 |
| **KEY3** (P1.5) | 风速 -1 档 | 蜂鸣一声, 风扇减速 |
| **KEY4** (P1.4) | 短按=切换模式, 长按≥2秒=设置/取消定时 | 短按一声, 长按两声 |

#### 红外遥控器按键功能

| 遥控器按键 | 功能 | 按下后反应 |
|:---:|---|---|
| **POWER (电源)** | 启动 / 停止风扇 | 蜂鸣一声, 风扇启/停, D1 LED 亮/灭 |
| **MODE (模式)** | 切换风类模式 | 蜂鸣一声, 模式切换: 常风→自然风→睡眠风→常风 |
| **VOL+ / +** | 风速 +1 档 | 蜂鸣一声, 风扇加速 (最高5档) |
| **VOL- / -** | 风速 -1 档 | 蜂鸣一声, 风扇减速 (最低1档) |
| **数字 1** | 直接选 1 档 (微风) | 蜂鸣一声, 风扇最慢 |
| **数字 2** | 直接选 2 档 | 蜂鸣一声 |
| **数字 3** | 直接选 3 档 (中速) | 蜂鸣一声 |
| **数字 4** | 直接选 4 档 | 蜂鸣一声 |
| **数字 5** | 直接选 5 档 (强风) | 蜂鸣一声, 风扇最快 |
| **PLAY (播放)** | 设置/取消定时 | 蜂鸣一声, 默认定时30分钟, 再按取消 |
| **PREV (快退)** | 定时 -5 分钟 | 蜂鸣一声, 最短为0(取消) |
| **NEXT (快进)** | 定时 +5 分钟 | 蜂鸣一声, 最长为60分钟 |
| **数字 0** | 取消定时 | 蜂鸣一声, 清除定时 |

### 2.2 Proteus 仿真版 (main_proteus.c — 纯按键控制)

#### 按键功能 (仿真版4个按键全部可用)

| 按键 | 操作方式 | 功能 | 按下后反应 |
|:---:|---|---|---|
| **KEY1** (P1.7) | 短按 | 启动 / 停止风扇 | 蜂鸣器响一声, 风扇启/停, D1 LED 亮/灭 |
| **KEY2** (P1.6) | 短按 | 风速 +1 档 | 蜂鸣一声, 风扇加速 (1→2→3→4→5→1循环) |
| **KEY3** (P1.5) | 短按 | 风速 -1 档 | 蜂鸣一声, 风扇减速 (5→4→3→2→1→5循环) |
| **KEY4** (P1.4) | **短按 (<2秒)** | 切换风类模式 | 蜂鸣一声, 模式切换: 常风→自然风→睡眠风→常风 |
| **KEY4** (P1.4) | **长按 (≥2秒)** | 设置 / 取消定时 | 蜂鸣**两声**(区分于短按), 默认30分钟; 再长按取消 |

> KEY4 长按判断: 按住 KEY4 不放, 约2秒后蜂鸣器发出连续两声, 说明定时已设置/取消。松手即可。

## 三、LCD1602 显示说明

### 3.1 风扇关闭时

```
┌────────────────┐
│  FAN STOPPED  │  ← 第1行: 风扇已停止
│Timer:OFF       │  ← 第2行: 未设置定时
└────────────────┘
```

### 3.2 风扇运行时 — 常风模式

```
┌────────────────┐
│S:3 NORMAL  ON  │  ← 第1行: Speed=3档, Normal=常风模式, ON=运行中
│Timer:30m       │  ← 第2行: 定时剩余30分钟
└────────────────┘
```

**第1行字段解释:**
| 显示 | 含义 | 可能的值 |
|---|---|---|
| `S:` | Speed 风速 | — |
| `1` | 当前风速档位 | `1`(1ms通≈12%) / `2`(2ms≈25%) / `3`(4ms≈50%) / `4`(6ms≈75%) / `5`(7ms通≈88%) | |
| `NORMAL` | 风类模式名 | `NORMAL`(常风) / `NATURAL`(自然风) / `SLEEP`(睡眠风) |
| `ON` | 风扇运行状态 | `ON`(运行中) 或 不显示(关闭时) |

**第2行字段解释:**
| 显示 | 含义 |
|---|---|
| `Timer:OFF` | 未设置定时, 风扇会一直运行 |
| `Timer:05m` | 定时剩余 5 分钟后自动关闭 |
| `Timer:30m` | 定时剩余 30 分钟后自动关闭 |

### 3.3 风扇运行时 — 自然风模式

```
┌────────────────┐
│S:4 NATURAL ON  │  ← Speed=4档, Natural=自然风模式
│Timer:OFF       │
└────────────────┘
```
- 自然风模式下, 转速会**忽大忽小**自动变化 (每3秒自动调整)
- D2 LED 会随速度变化闪烁

### 3.4 风扇运行时 — 睡眠风模式

```
┌────────────────┐
│S:5 SLEEP   ON  │  ← Speed=5档, Sleep=睡眠风模式
│Timer:25m       │  ← 设了定时(但不强求, Timer可OFF)
└────────────────┘
```
- 睡眠风模式下, 转速**每10分钟自动降一档**
- 如: 5档→10分钟后→4档→10分钟后→3档→...→1档
- D2 LED 随 PWM 周期闪烁

### 3.5 定时到零自动关机

当 LCD 第2行显示的定时从 `XXm` 倒数到 `00m` 后:
- 蜂鸣器不响 (静默关机)
- 风扇自动停止
- LCD 显示变回 `FAN STOPPED`
- D1 LED 熄灭

## 四、三种风类模式详解

### 4.1 常风模式 (NORMAL)
- **行为**: 以恒定转速运行, 转速 = 你设定的档位
- **LED**: D1 亮, D2 常亮
- **适用场景**: 需要稳定风力时

### 4.2 自然风模式 (NATURAL)
- **行为**: 每3秒自动在基础档位附近 ±2 档波动
  - 40% 时间: 保持基础档位
  - 20% 时间: 比基础档低1档
  - 20% 时间: 比基础档高1档
  - 10% 时间: 比基础档低2档
  - 10% 时间: 比基础档高2档
- **LED**: D1 亮, D2 根据换档节奏闪烁
- **适用场景**: 模拟户外自然风, 忽大忽小更舒适

### 4.3 睡眠风模式 (SLEEP)
- **行为**: 从当前档位开始, 每10分钟自动降一档, 直到1档
- **LED**: D1 亮, D2 随 PWM 周期闪烁 (形成"逐渐减弱"的视觉效果)
- **适用场景**: 睡前设置, 风扇慢慢减速, 配合定时关机会更好

## 六、Proteus 仿真操作步骤

1. 打开 Proteus 8 Professional
2. 新建项目, 选择默认模板
3. 按「一、Proteus 仿真电路」的元件清单放置所有元件
4. 按接线表连接所有电路
5. 双击 AT89C52, 在 Program File 中选择编译生成的 `main_proteus.hex`
6. 双击 AT89C52, 设置 Clock Frequency 为 `11.0592MHz`
7. 点击左下角 ▶ 运行仿真
8. 按 KEY1 启动风扇 → 观察 LCD 显示和直流电机转动
9. 按 KEY2 / KEY3 调节风速 → 观察转速变化
10. 按 KEY4 (短按) 切换模式 → 观察 LCD 模式名变化
11. 按住 KEY4 (>2秒) 设置定时 → 听蜂鸣两声, LCD 显示 Timer:30m
12. 观察定时倒数, 到达0后风扇自动停止

### 调试提示

- 如果 **LCD 无显示**: 检查 P0 数据线、RS/RW/EN 连线, 调一下 VEE 脚的对比度电位器
- 如果 **直流电机不转**: 检查 P1.0/P1.1 到 L298.IN1/IN2 连接, L298 的 ENA→+5V, VS→+12V, VSS→+5V 是否全部正确
- 如果 **按键无反应**: 检查上拉电阻 (10KΩ → VCC) 是否连接正确
- 如果 **LED 不亮**: 检查方向 (阳极→220Ω→引脚, 阴极→GND)
- 如果 **定时到零电机还在转**: 这是仿真速度问题, 50ms节拍在仿真中可能不准确
