Proteus 仿真操作步骤

1打开 Proteus 8 Professional，新建项目。
2点击左侧 Component Mode → 点 P，搜索放置所有元件（见清单）。
3参考接线图和接线表，按物理引脚号连接所有导线。
4双击 AT89C52 → Program File 选择 main_proteus.hex。
5Clock Frequency 设为 11.0592MHz。
6点击 ▶ 运行仿真 → 按 KEY1 启动风扇。
7KEY2/KEY3 调风速；KEY4 短按切换模式；KEY4 长按≥2秒设置/取消定时。

常见问题

现象排查
LCD 无显示检查 P0 数据线(pin39→32→LCD pin7→14)、RS/RW/EN 连线；VEE(pin3) 接地或接电位器调对比度
LCD 乱码晶振频率是否设为 11.0592MHz（不是 12MHz）；EA(pin31) 是否接 VCC
电机不转检查 P1.0~P1.3(pin1→4) 连接；COM 接 +5V；延时值是否太小（1档30ms 足够慢）
按键无反应检查 10K 上拉电阻到 VCC；引脚号是否正确（KEY1=pin11, KEY2=pin10 等）
LED 不亮共阳接法：LED 阳极经 220Ω 接单片引脚，阴极接 GND，低电平点亮
蜂鸣器不响P2.5(pin26) 同时驱动蜂鸣器和 LCD_RW，LCD 写操作关中断时蜂鸣器会受影响
定时不准Proteus 仿真速度可能影响 50ms 节拍精度，可用断点调试确认定时器 ISR 是否正常触发
